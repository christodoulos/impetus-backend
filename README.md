## Description

This is the [Digital Twins of Athens](https://atticadt.uwmh.eu/v2) backend application.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
